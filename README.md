# 🚀 Flask Tailwind Portfolio: Academic & Professional Showcase

A personal branding platform and portfolio built with **Flask** and **Tailwind CSS**. 

## 🌟 Key Highlights
- **Research Oriented:** Showcasing scientific publications in Neural Networks and SVR.
- **Dynamic Localization:** Multi-language support (ID/EN) for global reach.
- **Interactive UI:** 3D-rotated gallery and animated components using Tailwind CSS.
- **Responsive Architecture:** Fully optimized for all device breakpoints.

## 🛠 Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** Tailwind CSS, JavaScript (Vanilla)
- **Deployment Ready:** Configured with clean project structure for easy scaling.

## 📖 Features
- **Hero Section:** Personal branding with dynamic wave animation.
- **Publication Section:** Integrated links to scientific journals (e.g., Jurnal INFOTEL).
- **Project Gallery:** Visual display of software and research projects.
- **Contact Integration:** Direct WhatsApp and professional social media links.
